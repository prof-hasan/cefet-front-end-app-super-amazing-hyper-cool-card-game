# Jogos de Cartas (Inspiração: Marvel SNAP)

### Base
Jogo de cartas baseada em turnos, onde os jogador(es) começa com um “deck” inicial, e vão alterando entre cartas aleatoriamente selecionadas.
      
### Estrutura
Página inicial do site com principais informações, como: Instruções de como jogar, link para a loja de cartas, modos de jogo, e botão para o jogo. Após apertar o botão iniciar, o jogador(es) serão redirecionados para o inicio do jogo. 

### Extras

- Players: Estamos estudando qual o método de implementação mais conveniente para o jogo:
    - Multiplayer local, utilizando um servidor com base em WebSockets;
        - Criação de uma IA como adversária;
- Game Engine: planejamos a utilização de uma game engine como Phaser.js;
